# クエスト課題回答

初回クエスト（シェルスクリプトを用いてパスワードマネージャーを作る）の回答になります。

## step-1
ステップ１の課題は`step-1`ディレクトリ内のシェルスクリプトに記載しています。

### 操作方法
①：`cd`コマンドでシェルスクリプトが存在するファイルに移動します。
```bash
cd step-1
```

②：シェルスクリプトを実行します。
```bash
./password_manager.sh
```

③：対話形式で入力が求められるので情報を入力していきます。
```bash
パスワードマネージャーへようこそ！
サービス名を入力してください：sample
ユーザ名を入力してください：user
パスワードを入力してください：pass
Thank you
```

正常に処理が実行されると`passwords_file.txt`というテキストファイルが作成されます。ここに入力情報が保存されます。

↓ テキストファイルの中身
```
sample:user:pass
```

## step-2
ステップ２の課題は`step-2`ディレクトリ内のシェルスクリプトに記載しています。

### 操作方法
①：シェルスクリプトがあるディレクトリへ移動します。
```bash
cd step-2
```

②：シェルスクリプトを実行します。
```bash
./password_manager.sh
```

③：実行が始まると以下のように初期画面が表示され、対話形式で行いたい操作を選択・実行することができます。
```
パスワードマネージャーへようこそ！
次の選択肢から入力してください(Add Password/Get Password/Exit)：
```

機能としては、新たにデータを保存する`Add Password`、登録データを参照する`Get Password`、処理を終了する`Exit`の3つが存在します。
※　存在しない機能を入力した場合は再入力が求められます。
```bash
次の選択肢から入力してください(Add Password/Get Password/Exit)：@1234
入力が間違えています。Add Password/Get Password/Exit から入力してください。
```

### データの新規登録
`Add Password`を選択するとサービス名とユーザー名、パスワードの入力を求められるので適宜入力すると`passwords_file.txt`に内容が保存されます。
```bash
次の選択肢から入力してください(Add Password/Get Password/Exit)：Add Password
サービス名を入力してください：sample
ユーザー名を入力してください：user
パスワードを入力してください：pass


パスワードの追加は成功しました。
```

### 登録データの参照
`Get Password`を選択すると登録情報を検索することができます。サービス名を入力すると合致するデータがあれば詳細情報を表示します。
```bash
次の選択肢から入力してください(Add Password/Get Password/Exit):Get Password
# 照合に成功した場合
サービス名を入力してください：sample
サービス名: sample
ユーザー名: user
パスワード: pas

# 照合に失敗した場合
サービス名を入力してください：hoge
そのサービスは登録されていません。
```

### シェルスクリプトの終了
トップ画面から`Exit`を選択するとシェルスクリプトを終了することができます。
```bash
次の選択肢から入力してください(Add Password/Get Password/Exit):Exit
# 終了してコマンドが実行できるようになる
```
